# Challenger Clyvo Vet API

API RESTful desenvolvida em ASP.NET Core.

## Tecnologias

- ASP.NET Core
- Entity Framework Core
- Oracle
- Swagger

## Como executar

1. Clone o projeto
2. Configure appsettings.json
3. Execute migrations
4. Rode a aplicação

## Endpoints

GET /api/pets

POST /api/pets

PUT /api/pets/{id}

DELETE /api/pets/{id}
